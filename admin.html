💡 How It Works

Changes made here update the site_content table in Supabase.

The homepage reads this table on load — so every visitor sees your latest content within seconds.

If Supabase is unavailable, the site falls back to its built-in defaults.

Required Supabase tables:
site_content (key, value)
site_slides (id, url, type, title, subtitle, caption, tag_text, tag_type, cta_text, cta_url, overlay_color, sort_order, active)
site_ticker (id, text, type, sort_order, active)
Storage bucket: site-media (public)
